Cómo ejecutarlo

No hace falta instalar nada más que el JDK: las librerías (Lombok y JUnit) ya vienen en la carpeta lib/ del repositorio.

◆ Requisitos

  • JDK 17 o superior (comprueba con java -version).
  • Una terminal que soporte colores ANSI (cualquier terminal moderna de Linux, macOS o Windows).
  • git para clonar el repositorio (opcional: puedes descargar el ZIP).

◆ 1. Clonar

git clone https://github.com/RenzoRamosDEV/Juego-X-Turnos.git
cd Juego-X-Turnos

◆ 2. Compilar y jugar

La forma más simple es el script incluido:

./run.sh

O paso a paso, si prefieres:

# reunir las fuentes y compilar con Lombok
find src -name "*.java" > sources.txt
javac -cp lib/lombok.jar -d out @sources.txt && rm sources.txt

# lanzar el juego
java -cp out com.tuio.juegoxturnos.Main

◆ 3. Ejecutar los tests

./test.sh

Compila el código y las pruebas y lanza los 60 tests con JUnit 5. Es lo mismo que corre la CI en cada push.

◆ Los scripts

run.sh

Compila las fuentes (con Lombok) y arranca el juego. Un solo comando para jugar.

test.sh

Compila código y tests y ejecuta la batería completa de JUnit 5, mostrando el árbol de resultados.

Importante: el juego es interactivo (lee lo que escribes) y usa colores y limpieza de pantalla. Córrelo en tu terminal, no redirigido a un archivo ni por una tubería.
